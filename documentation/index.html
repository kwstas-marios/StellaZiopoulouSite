About

Soundlab is a clean and professional site template, perfect for Music Event, Live music and ticket music web sites. Built on the Advanced Bootstrap 3 framework with HTML5 / CSS3. Its responsive too - will fit great on all devices. More features with , and much more gives you more space to work with and also with a few lines of code.

Clean coded and Easy updation are the key features. Go ahead to know more.

CSS files and structure

I have used 12 CSS files in this theme.

  1. main.css - template styl file.
  2. bootstrap.css - Bootstrap css.
  3. font-awesome.min.css - Fontawesome Icons.
  4. superslides.css - A fullscreen slider for jQuery
  5. owl.carousel.css - Contains all of the specific stylings for Owl Carousel
  6. owl.transitions.css - Contains all of the specific stylings for Owl Carousel
  7. flickity.min.css - Contains all of the specific stylings for Flickityl
  8. TimeCircles.css - Contains all of the specific stylings for TimeCircles plugin
  9. swipebox.css - Contains all of the specific stylings for Swipebox plugin
  10. intro-slider.css - Contains all of the specific stylings for Intro slider
  11. animate.min.css - Contains all of the specific stylings for Intro slider
Main.css file contains all of the specific stylings for the page. You can get more check more style files in styles folder. There is demo.css for insert image in our template.

Javascript

This theme imports Twenty Javascript files.

  1. modernizr.custom.js - Modernizer plugin.
  2. jquery.js - To Enable jQuery.
  3. jquery.superslides.min.js - To Enable SuperSlider.
  4. flickity.pkgd.js - To enable Flickity plugin.
  5. audio.min.js - Audio plugin.
  6. twitterFetcher_min.js - Twitter Feet plugin.
  7. isotope.pkgd.min.js - Isotope Plugin.
  8. jquery.swipebox.js - Lightbox Plugin.
  9. TimeCircles.js - TimCircles plugin.
  10. owl.carousel.min.js - Carousel plugin.
  11. jquery.parallax.min.js - Parallax plugin.
  12. discography.js - Album plugin.
  13. news.js - News post jquery.
  14. main.js - Main jquery file.
  15. bootstrap.js - Bootstrap script file.
  16. intro-slider.js - Slider script.
  17. countdown.js - Countdown script.
